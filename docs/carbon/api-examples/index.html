API examples

Compute operational energy and co2eq in one hour​

Json body exemples when using /operation based API. Have a look to our Carbon calculator API documentation for more details.

Note that all exemples provides the estimation of operational energy and co2eq during one hour, assuming a CPU load of 50%, a GPU load of 50% and a data transfer during the hour of 4294967296 Bytes.

AWS examples​

Compute instance with 4GB storage​

Use POST /operation/aws.

{
    "instance": {
        "type": "t2.medium",
        "region": "ap-northeast-3",
        "storage": {
            "size": "4294967296",
            "type": "sc1"
        }
    },
    "metrics": {
        "cpuLoad": 0.5,
        "dataTransfer": 4294967296
    }
}

GPU instance with 32 GB storage​

Use POST /operation/aws.

{
    "instance": {
        "type": "g3.8xlarge",
        "region": "eu-west-2",
        "storage": {
            "size": "34359738368",
            "type": "sc1"
        }
    },
    "metrics": {
        "cpuLoad": 0.5,
        "gpuLoad": 0.5,
        "dataTransfer": 4294967296
    }
}

Custom infrastructure​

RAW (user-defined) mode compute instance​

Use POST /operation/raw.

{
    "instance": {
        "cpu": {
            "type": "Cascade Lake",
            "count": 1
        },
        "ram": {
            "size": "17179869184"
        },
        "storage": {
            "size": "34359738368",  
            "rotational": true,
            "replication": "1"
        },
        "energy": {
            "pue": "1.4",
            "carbon": "0.0000141944" 
        }
    },
    "metrics": {
        "cpuLoad": 0.5,
        "dataTransfer": 4294967296
    }
}
