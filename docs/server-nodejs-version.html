Select version of Node.js to use

To select which version of Node.js to deploy and run you have use a configuration file named warp.config.js in the directoy where you deploy.

In that file you can specify the Node.js version to use using a semver range to specify the engine version.

You have to set engine.node property in the warp.config.js file.

module.exports = {
    engine: {
        node: 12
    }
}

The chosen version will be the latest LTS version in your range. If none are LTS, then it will be the latest version in your range.

If engine.node is not defined the latest LTS version of node is used by default.
