Get module execution logs

With the CLI of the SDK, you can access the execution logs of your module.

Get the list of deployments

You must have selected a project and an environment.

Then to get the list of all deployments let's use that command:

npx warp deployment list

The command will dump you something like that:

    name                                  description   state       date                        
    deployment-2021-08-02T10:13:05.817Z                 uploading   Mon Aug 02 2021, 12:15:11                             
    deployment-2021-08-06T14:46:04.890Z                 running     Fri Aug 06 2021, 16:48:39

Get logs of module running a deployment

To fetch the module logs use the name of the deployment you want to get.

For example

  npx warp deployment logs deployment-2021-08-06T14:46:04.890Z

To get realtime update of module logs

To fetch the module logs and follow in realtime, use the previous command with --follow flag.

For example

  npx warp deployment logs --follow deployment-2021-08-06T14:46:04.890Z
