Introduction

Welcome to our computing platform, the simplest one to develop and the most flexible to run.

Principle is simple: write modules, we run them where you choose.

What we solve

Our platform solves major issues when developing/deploying and run on the cloud.

Time to learn cloud to be able to start developing/deploying and run is hudge

In general it takes 2-3 months to someone to be able to start developing/deploying and run on the cloud. Thanks to our Module as a Service approach, you code Node.js modules, not servers or functions that will run as a Service. With our approach JavaScript developer can develop without coding http layer (generated automatically with our compiler), routing, security management, provisioning, endpoint setup... Just know how to code a Node.js module and your're ready to go in seconds.

Developing/deploying a cloud code is complex, impacting release/features roadmap

We offer free tools and templates to develop faster than ever and in the most simple way. With our template builder define your templates and start developing in seconds. With our online playground, develop or test some module snippet in a frontend instantly. With our SDK discover live reload frontend and backend to develop on your laptop with minimum coding time; Our SDK is agnostic in terms of bundler and application frameworks. With our CLI create easily your CI/CD or deploy right from your laptop in one command line, our IT automation takes care of provising, setup, deployment and run of your modules.

Possibilites to run are limited

we provide a cloud resource store you can use transparently to run your codes. We provide the most important choice of run on the market. In one place, and in one click, choose your cloud resource provider, your cloud resource hardware configuration, your location/region according to your needs. We manage the provisioning, setup and run for you transparently.

Changing from one cloud configuration to another is hard, sometimes impossible

Want to publish your SaaS on a cloud marketplace ? Want to change of hardware configuration to run faster ? Want to support a larger traffic? Want to move your proceesing closer to a database or to your new users (going from France to US for example)? Using our platform change anytime of cloud resource transparently for your users in one click. Follow your application/service requirements anytime per module.

What we provide

  • MIT Templates
  • A template builder
  • An online fullstack playground
  • a SDK to install for linux/windows/mac
  • a static hosting with custom domains, and https certificate management
  • a cloud resource store to select and change the cloud resource to use in several clicks.
  • access to a best effort cloud resource in all plans, so you can start easily using our platform.

Develop for free, pay only when you deploy and run

We dont charge for all development steps (templates, template builder, SDK, the online playground) so you can develop your modules without constraints.

Just pay when you use cloud resources to deploy or to run. We provide a free community plan to give access to deploy and run for free also. Just create an account here and you're ready to go.

  • a static webapp
  • a dynamic webapp
  • your SaaS implementation, front/back office
  • computing or batch processing microservices
  • any processing you have to do
