Introduction

ScaleDynamics is a fully-managed next generation computing platform. ScaleDynamics handles all the complexity of deploying, managing, and healing your deployments, with maximum simplicity and cloud agility in mind.

On our platform you write Node.js modules and we run them as-a-Service. Get details on Module-as-a-Service and use cases. We provide you a set of tools for you to develop in the simplest way, without cloud knowledge. If you know Node.js, JavaScript or TypeScript you're ready to go!

On our Cloud Store in few clicks you can choose the cloud resource to run your modules. We provide a lot of different configuration (performance, cloud providers, regions, price) so you can select the best one for your module need. And with us you can change anytime of resources in few clicks, giving you the maximum cloud agility.

Our platform provides you integrated and advanced cloud management features. From remote log access, analytics, provisioning... you'll enter a new cloud experience.

Our platform provides you a complementary hosting for your static assets (html, css, js, images...) so you have all in place to code for the web.

Time to get started

  • Deploy your first module in less than 2 minutes
  • Access our full stack playground to test frontend and module snippets in real time
  • Use our template builder to create and download your own pre-configured project templates
  • Browse the cloud store to browse our catalog of resources
  • Download the sdk, start coding in live reload and deploy in one command line
