Import‚Äã

To use the cloud Node.js module from a Browser script, Node.js script or module, you need first to import the client module that was generated after build. To use it you have to import Warp engine first. The engine has to be initialize just once.

CommonJS (CJS)‚Äã

require("@warpjs/engine");

ES Modules (ESM)‚Äã

import "@warpjs/engine";

HTML script tag‚Äã

Include the Warp engine directly in an HTML file:

<script src="https://cdn.scaledynamics.com/warp-engine/4.0.19/engine-web-prod.js" type="text/javascript"></script>

Pay attention to using the same engine version that you use to develop the Node.js server part. Please change the version in the CDN url. Find current version of SDK thanks to the npx warp version command.
