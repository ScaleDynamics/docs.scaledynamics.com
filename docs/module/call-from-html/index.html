Call from browser using HTML

To use the cloud Node.js module from a Browser script, you need first to import the client module that was generated after build. You also need to import the Warp Engine that will provide you access to the functions of the module you want to use in your script.

Import the client module​

In order to use it from HTML, the client module should have been build in umd format.

In your html, you have to import it using a script:

<script src="client_module.js"></script>

Get access to the module​

Now you loaded it, you need to get access to it. 

<script>
  const MicroService = window['module-client']
</script>

Load Warp engine​

Next step is to load the Warp engine. Warp engine provides you the way to call functions of your Node.js module using its client. You have 2 ways to load it: using loadEngine client module function or using a Warp engine module dependency.

Using loadEngine()​

A client-module provides a loadEngine function. That function will load the latest compatible release of the Warp engine for the client module build. We recommend using this way because you never have to manage the Warp engine version. 

<script>
  const MicroService = window['module-client']
  MicroService.loadEngine().then( ... )
</script>

or

<script>
  const MicroService = window['module-client']
  async function start() {
    await MicroService.loadEngine()
  }
  start()
</script>

Using an engine script​

In case you cannot use or adapt your code to be asynchronous and use the loadEngine way, you can directly import the engine from an url before loading the client module. At each release of a warp engine we push it on a CDN. You then have to load the right engine version which should match with the Warp version you used when building the client module. 

<script src="https://cdn.scaledynamics.com/warp-engine/5.0.0/engine-web-prod.js"></script>
<script src="client_module.js"></script>

Note that you have to manage yourself the dependency of the Warp engine that must be compatible with your client module build.

Get client module functions and use them​

Now you have loaded the client module and warp engine you can set your functions and call them.

<script>
const MicroService = window['module-client']
MicroService.loadEngine().then(()=> {
  const { hello } = new MicroService()  
  hello('World').then((message) => {  
    console.log(message)
  })  
})
</script>

Full example​

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width,initial-scale=1" />
    <title>You Node.js module use</title>
  </head>
  <body>
    ...
    <script src="client_module.js"></script>
    <script>
      const MicroService = window['module-client']
      MicroService.loadEngine().then(()=> {
        const { hello } = new MicroService()  
        hello('World').then((message) => {  
          console.log(message)
        })  
      })
    </script>
  </body>
</html>
