Call from browser using a script

To use the cloud Node.js module from a Browser script, you need first to import the client module that was generated after build. You also need to import the Warp Engine that will provide you access to the functions of the module you want to use in your script.

Import the client moduleâ€‹

In order to use it from HTML, your Node.js module should have been build in umd format.

Then you can import it with its path:

import MicroService from './client_module.js'

Load Warp engineâ€‹

Warp engine provides you the way to call functions of your Node.js module using its client. You have 2 ways to load it: using loadEngine function or using a Warp engine module dependency.

Using loadEngine()â€‹

A client module provides a loadEngine function. That function will load the latest compatible release of the Warp engine for the Node.js module build. We recommend using this way because you never have to manage the Warp engine version. 

import MicroService from './client_module.js'
MicroService.loadEngine().then( ... )

or

import MicroService from './client_module.js'
async function start() {
  await MicroService.loadEngine()
}
start()

Using an importâ€‹

In case you cannot use or adapt your code to be asynchronous, you can directly import the engine, before requiring the client module.

Of course you need to install it before using it

npm install @warpjs/engine

Then requires it

import('@warpjs/engine')
import MicroService from './client_module.js'

Note that you have to manage yourself the dependency of the Warp engine that must be compatible with your Node.js module build.

Get client module functions and use themâ€‹

Now you have loaded the client module and warp engine you can set your functions and call them.

const { hello } = new MicroService()  
// Call the function of your Node.js module
hello('World').then((message) => {  
  console.log(message)
})

Full exampleâ€‹

import MicroService from './client_module.js'
MicroService.loadEngine().then(()=> {
  const { hello } = new MicroService()  
  hello('World').then((message) => {  
    console.log(message)
  })  
})
