Call from Node.js

To use the cloud Node.js module from a Node.js script or another Node.js module, you need first to import the client module that was generated after build. You also need to import the Warp Engine that will provide you access to the functions of the module you want to use in your script.

Import the client module‚Äã

In order to use it from HTML, the client module should have been build in umd format.

Then you can requires it with its path:

const MicroService = require('./client_module.js')

Load Warp engine‚Äã

Warp engine provides you the way to call functions of your client module. You have to load it using a Warp engine module dependency.

Using a require‚Äã

In case you cannot use or adapt your code to be asynchronous, you can directly import the engine, before requiring the client module.

Of course you need to install it before using it

npm install @warpjs/engine

Then requires it

require('@warpjs/engine')
const MicroService = require('./client_module.js')

Note that you have to manage yourself the dependency of the Warp engine that must be compatible with your client module build.

Get client module functions and use them‚Äã

Now you have load the client module and warp engine you can set your functions and call them.

const { hello } = new MicroService()  
// Call the function on the client module
hello('World').then((message) => {  
  console.log(message)
})

Full example‚Äã

require('@warpjs/engine')
const MicroService = require('./client_module.js')
const { hello } = new MicroService()  
hello('World').then((message) => {  
  console.log(message)
})
