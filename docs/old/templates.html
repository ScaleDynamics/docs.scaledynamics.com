Getting started from templates

The best way to start on our platform is to use our project templates. They are preconfigured and ready to use for developing in live reload, deploy and make it live on a url on our platform.

Frontend and backend templates:

  • MERN Stack template

MEAN stands for MongoDB, Express, React and Node.js Full Stack

  • MEAN Stack template

MEAN stands for MongoDB, Express, Angular and Node.js Full Stack

  • MEVN Stack template

MEVN stands for MongoDB, Express, Vue and Node.js Full Stack

  • Vanilla MongoDB template

Stands for MongoDB, Express, Vanilla HTML5 and Node.js Full Stack

Microservice template:

  • Microservice

all templates are available on Github

You can also use our Template Builder tool to create your templates with the options you want (frontend/backend, microservices projects).
