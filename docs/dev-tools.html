Dev Tools

We provide you development tools to code, deploy and run your modules in the most simple way.

Note that all tools are free to use (templates, template builder, SDK, the online playground) so you can develop your modules without constraints. We also provide a free developer plan to give access to deploy and run for free also for non commercial use. Just create an account here and you're ready to go to use all the platform.

SDK

Our is available on npm, and available on Windows/Mac/Linux.

npm install warp

It provides you a CLI you can use to create your projects, develop in live-reload on your laptop, deploy from your laptop, build your CI/CD workflow, and manage your module runs.

Access SDK introduction for more details.

Console

The console is used to manage organizations, users, projects, environments and cloud resource settings. You are also able to follow all deployments you did.

Access it here

Note that you will need to create an account to access it.

Realtime playground

This is the first online playground that allows coding directly in the browser for both front-end JavaScript and back-end Node.js, showing results in real-time with zero configuration. It provides you every you need to show front-end and back-end working together in one online place.

Our playground uses our Direct call module as a Service coding approach. No need to create HTTP calls, just create a node.js module and call its exported functions right in JavaScript.

The playground provides you a way to browse, create or fork full stack code snippets. A Snippet is composed of one HTML file, one CSS file and one JavaScript file for the browser side, and one Node.js module file with its npm module dependencies.

Each time you click on the "Run" button, the Node.js module is deployed on a real physical server on the cloud (currently on Azure). All is managed for you from the deployment or accessing the backend logs to debug your code. The preview gives you the result of your frontend+backend execution right in your browser.

Play with it.

Template builder

With our template builder create pre-configured project templates ready to use. From FullStack or microservice projects, get your template an start coding in seconds.

Build your first template.

Our cloud store provides a catalog of preconfigured ressources available to run your modules. We provide a lot of different configuration (performance, cloud providers, regions, price) so you can select the best one for your module need in few clicks.
